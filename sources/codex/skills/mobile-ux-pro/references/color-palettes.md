## 使用方法

設計時選擇色板步驟：
1. 確認 App 類別（Social/Health/Finance/Gaming/Productivity/Commerce/Media/Education/Lifestyle/Travel）
2. 在該類別中選 2-3 個候選色板（根據品牌語氣/目標情感）
3. 用 CSS Variables 實現（見 SKILL.md 模板）
4. 驗證：Text on BG ≥ 4.5:1（AA），Large Text ≥ 3:1

Dark Mode 轉換規則：
- BG → #121212 或 #0D0D0D
- Surface → #1E1E1E 或 #252525  
- Text → #E0E0E0 或 #F5F5F5
- Primary 保持，但需重新驗證對比度
