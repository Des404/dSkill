# 台灣本地化規範（TW Locale）

載入條件：「台灣」「TW」「台灣用戶」

## 語言規範
- 繁體中文（與 HK 同字形，但部分用字不同：e.g. HK「係」vs TW「是」）
- 字體：Noto Sans TC / iOS: -apple-system
- 正文 ≥ 16px，行距 1.7

## 主流平台（2025）
LINE（最重要，日常通訊）· Facebook · Instagram · YouTube · Dcard（年輕人論壇）· PTT（老一輩）

## 支付方式
LINE Pay（最普及）· 街口支付（Jkos）· Pi 錢包 · Apple Pay · 信用卡 · 超商代收

## 分享渠道優先
① LINE 好友/群組（最高轉化）② Facebook ③ Instagram ④ 複製連結

## 視覺偏好
- 可愛/療癒風格比 HK 更受歡迎
- LINE 貼圖文化影響 UI 設計（表情豐富）
- 粉色/薰衣草紫受女性用戶歡迎
- 年輕族群（18-30）重視 IG-worthy 設計美感

## 特有設計考量
- LINE 分享按鈕必須第一位（台灣分享以 LINE 為絕對主導）
- Dcard 評論文化：匿名、重視星級評分
- 超商 QR Code 支付場景需考慮
- 繁中字形注意：台灣用字習慣與香港略有不同

## 法規
個人資料保護法（PDPA）
