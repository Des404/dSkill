# 分享功能模式庫

## 分享工程鏈
用戶有動機 → 分享超簡單（≤2步）→ 內容值得被看 → 收到者會安裝

## 動機金字塔（由強到弱）
1. 社交貨幣（展示自己）→ 成就卡、排名、個人化結果
2. 利益驅動 → 雙向獎勵邀請（Dropbox 模型）
3. 情感連結 → Spotify Wrapped、健身紀錄
4. 實用資訊 → 食譜、攻略
5. FOMO → 「你朋友在用」

## 香港渠道優先順序
① WhatsApp 個人/群組（轉化最高，必須第一）
② IG Story（18-35 歲，曝光廣）
③ Facebook（35+ 歲）
④ 複製連結（萬能後備）
⑤ 原生 Share Sheet（iOS/Android 系統）

## 分享卡規格
- IG Story：1080×1920px（9:16）
- WhatsApp：1080×1080px（1:1）
- 必要元素：App Logo（小）、核心數據（大字）、引好奇文案（≤15字）、下載 QR Code
- 禁止：純文字、無數據、無品牌標識

## 觸發時機
✅ 完成任務慶祝畫面、達里程碑、生成個人化結果、排名超過朋友
❌ 剛進 App、用戶操作中、失敗/錯誤狀態

## 流程（2步最佳）
[完成任務] → 自動生成分享卡 → 點「分享到 WhatsApp」→ 完成

## 文案預填
- 「我剛係呢個 App 完成咗 XXX，你試下！」（港式口語效果最好）
- 提供搞笑版/正式版/簡短版 3 個選項

## Referral 雙向激勵
- 邀請方 + 被邀請方各得具體價值（不是積分）
- 「你邀請一個朋友，雙方各獲 7 天 Premium」
- 技術：Branch.io 或 AppsFlyer OneLink（Deep Link + 歸因）
- 連結有效期：30 天

## K-factor 計算
分享 → 安裝 → DAU 增加 → 廣告曝光 → 廣告收入
K-factor = 每用戶平均邀請數 × 轉化率
K > 1 = 病毒式增長
